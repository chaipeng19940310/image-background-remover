# Image Background Remover — MVP 需求文档

> 版本：v0.1 | 日期：2026-03-19 | 状态：草稿

---

## 1. 产品概述

### 1.1 产品定位

一个轻量级在线图片背景去除工具。用户上传图片，一键获得透明背景的 PNG，无需注册、无需安装。

### 1.2 目标用户

- 电商卖家（商品图抠图）
- 设计师（快速处理素材）
- 普通用户（证件照、头像处理）

### 1.3 MVP 核心目标

验证用户对该工具的需求和付费意愿，快速上线，低成本运营。

---

## 2. 功能范围

### 2.1 MVP 包含（Must Have）

| 功能 | 描述 |
| --- | --- |
| 图片上传 | 支持点击选择 + 拖拽上传，格式：JPG / PNG / WEBP |
| 背景去除 | 调用 remove.bg API 处理，返回透明背景 PNG |
| 结果预览 | 处理完成后展示 before / after 对比 |
| 图片下载 | 一键下载处理后的 PNG 文件 |
| 无存储 | 图片仅在内存中处理，不落盘、不入库 |
| 错误提示 | API 失败、文件格式错误、超大文件等给出友好提示 |

### 2.2 MVP 不包含（Out of Scope）

- 用户注册 / 登录
- 历史记录
- 批量处理
- 自定义背景替换
- 移动端 App

---

## 3. 用户流程

```
进入首页
  → 上传图片（拖拽 or 点击）
  → 显示原图预览 + Loading 状态
  → 调用后端 API → remove.bg 处理
  → 展示处理结果（before/after 对比）
  → 点击下载 PNG
  → （可选）重新上传新图片
```

---

## 4. 技术方案

### 4.1 技术栈

| 层级 | 技术选型 |
| --- | --- |
| 前端框架 | Next.js 14（App Router） |
| 样式 | Tailwind CSS |
| 背景去除 | remove.bg API |
| 部署 | Vercel |
| 语言 | TypeScript |

### 4.2 架构说明

- 前端直接与 Next.js API Route 通信
- API Route 作为中转层，持有 remove.bg API Key（不暴露到客户端）
- 图片以 FormData 形式在内存中传递，全程不落盘

```
浏览器 → Next.js API Route（/api/remove-bg）→ remove.bg API → 返回 PNG → 浏览器
```

### 4.3 文件结构

```
image-background-remover/
├── app/
│   ├── api/
│   │   └── remove-bg/
│   │       └── route.ts       # API 中转层
│   ├── page.tsx               # 主页面
│   └── layout.tsx
├── .env.local                 # REMOVE_BG_API_KEY=xxx
└── package.json
```

### 4.4 关键约束

| 约束 | 说明 |
| --- | --- |
| 文件大小 | 限制上传 ≤ 10MB |
| 支持格式 | JPG、PNG、WEBP |
| API Key 安全 | 仅在服务端 API Route 使用，不暴露前端 |
| 并发 | MVP 阶段不做队列，直接透传 |

## 5. 页面设计

### 5.1 首页（唯一页面）

**状态一：初始上传态**
- 居中上传区域，支持拖拽
- 提示文字："拖拽图片到此处，或点击上传"
- 支持格式说明

**状态二：处理中**
- 显示原图缩略图
- Loading 动画 + 文字提示"正在去除背景…"

**状态三：处理完成**
- Before / After 左右对比展示
- 下载按钮（醒目）
- "重新上传"按钮

**状态四：错误**
- 友好错误提示
- 重试按钮

## 6. 非功能需求

| 项目 | 要求 |
| --- | --- |
| 响应时间 | 处理结果 ≤ 10s（取决于 remove.bg） |
| 隐私 | 图片不存储，处理完即释放内存 |
| 浏览器兼容 | Chrome / Safari / Firefox 最新版 |
| 移动端适配 | 响应式布局，可用但非重点 |

---

## 7. 成功指标（MVP 验证）

| 指标 | 目标 |
| --- | --- |
| 上线时间 | 1 周内 |
| 用户完成率 | 上传后成功下载 ≥ 70% |
| 用户反馈 | 收集 20+ 条真实用户反馈 |
| API 成本 | 控制在可接受范围内（remove.bg 按次计费） |
